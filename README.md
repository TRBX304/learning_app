## My Question Bank | AI搭載オリジナル問題集作成アプリ

### 概要
教材から問題を自動生成し、弱点分析を行う学習支援Webアプリ。

### 主な機能
- AIによる自動作問
- 解答履歴の分析・弱点可視化

### 使用技術
- Frontend: HTML / CSS / JavaScript
- Backend: Supabase Edge Functions (TypeScript)
- Database: Supabase
- AI: GPT API

### 動作環境
- 最新のWebブラウザ（Chrome等）
